바닥 인식이 안되는 경우가 생겨서 공간에 대한 테스트를 진행한다.
- [ ] 벽이 완벽히 막히지 않은 경우를 테스트
- [ ] 벽을 다 막은 상태에서의 테스트
- [ ] 중간에 가구배치가 바뀌었을 때 테스트
- [ ] 바닥 투명하게 할지
