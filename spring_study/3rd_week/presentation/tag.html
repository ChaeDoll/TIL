&nbsp; : 공백 한 칸(Non-breaking Space) [ ]
&ensp; : 중간 스페이스 [ ]
&emsp; : 큰 스페이스 [ ]
&lt; : 부등호 기호 "~보다 적다(less than)"
&gt; : 부등호 기호 "~보다 크다(greater than)"
&amp; : 앰퍼샌드(ampersand) 기호 "그리고(and)"
&quot; : 쌍따옴표(") 기호
<div></div> : 박스 또는 레이어를 나누는 태그. 레이아웃에 주로 쓰임

<p></p> : 문단(paragraph)을 묶어주는 태그

<b> : 텍스트에 두꺼운(bold) 효과를 주는 태그
<i> : 텍스트에 기울임(italic) 효과를 주는 태그
<strong> : 텍스트에 강조(strong) 효과를 주는 태그
<em> : 텍스트에 약한 강조(emphasized) 효과를 주는 태그
<ins> : 텍스트에 삽입됨(insert) 효과를 주는 태그
<del> : 텍스트에 삭제(delete) 효과를 주는 태그
<s> : 텍스트에 취소(strikethough) 효과를 주는 태그 : del보다는 약한 의미
<u> : 텍스트에 밑줄(underline) 효꽈를 주는 태그 : 철자 오류 지적용
<sup> : 텍스트에 위첨자(superscript) 효과를 주는 태그
<sub> : 텍스트에 아래첨자(subscript) 효과를 주는 태그
<small> : 텍스트에 크기를 줄이는(small) 효과를 주는 태그
<br> : 줄바꿈(line break) 태그
<ht> : 가로줄(horizontal rule) 태그 ↓
<a href="link"></a> : 하이퍼링크(Anchor) 생성 태그
<img src="link" alt="description" title="information" height="pixel or %" width="pixel or %"> : 이미지(image) 태그
<video src="link"> : 비디오(video) 태그
<audio src="link"> : 오디오(audio) 태그
<progress value="now value" max="max value"> : 진행상황 막대
