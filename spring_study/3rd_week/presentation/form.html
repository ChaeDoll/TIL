form에 들어가는 요소들

<form> : 입력값을 받는 영역(form)을 지정. 여러개의 영역은 가능하지만, 영역 안에 영역은 안된다.
<input> : 입력값 요소(input)을 지정. type에 따라 다른 입력값을 보인다.
<textarea> : 여러 줄의 문자열 값을 받는다.
<button> : input의 button과 동일하다. 태그 내에서 HTML 지정이 가능해서 표현 폭이 넓다.
<datalist> : 검색어에 들어갈 목록을 지정한다.
<select> : 선택 목록 (하위 목록)
<fieldset> : 폼에 있는 요소를 그룹으로 묶어준다. 사각형 박스에 가둠.
<legend> : fieldset으로 묶어준 그룹에 제목을 지정하여 사각형 박스에 표기됨
