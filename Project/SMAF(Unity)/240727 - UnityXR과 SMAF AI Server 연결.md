- [x] 문제 해결
PC에서 테스트할 때는 잘 작동하는데 Build만하면 작동하지 않는 문제가 있었다. 처음에는 Android에 있는 권한 문제인가 싶어서 이것저것 많은 시도를 하였는데, Debug를 거치는 과정에서 애초에 서버로 통신이 가지도 않는다는 것을 알게되었다.
